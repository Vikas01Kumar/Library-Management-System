# Library Management System

This is a web-based application developed as a final year BCA project. The system is designed to automate and manage the primary operations of a library, such as maintaining book records, managing student memberships, and tracking the issuance and return of books.

### 🚀 Tech Stack Used:
* *Backend:* PHP
* *Database:* MySQL
* *Frontend:* HTML, CSS, JavaScript, Bootstrap 4.4.1
* *Server:* XAMPP (Apache & MySQL)

### 🔑 Key Features:
* *Admin Dashboard:* Total control over books, categories, authors, and student records.
* *Student Dashboard:* Students can view their profile and issued books.
* *Book Management:* CRUD operations (Add, View, Edit, Delete) for books and categories.
* *Issue & Return Tracking:* Efficient system to track which student has which book and manage due dates.
* *User Authentication:* Secure Login and Registration system for both Admin and Students.

### 🛠️ How to Run:
1. Download the project and place it in the htdocs folder of your XAMPP directory.
2. Open *phpMyAdmin* and create a new database named lms.
3. Import the provided lms.sql file into the database.
4. Open your browser and go to localhost/Library-Management-System/index.php.
